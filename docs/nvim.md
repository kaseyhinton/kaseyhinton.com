# NVIM


### Open tree 

```
ctr + n
```

### Focus tree

```
space + e
```

### Add file in tree

```
a
```

### Fix broken icons in neovim tree 

```
sudo vpm install nerd-fonts
```


### Remove old neovim

```
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.cache/nvim

sudo vpm remove nvim
```
